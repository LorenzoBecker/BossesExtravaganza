# Forge release notes draft

Bosses Extravaganza adds configurable extra vanilla bosses to SPT raids while preserving vanilla boss spawns.

Main features:

- Per-map, per-boss spawn chances.
- Configurable maximum extra bosses per raid.
- Zone validation against real spawnpoints before injection.
- Bot/BotPmc spawnpoint promotion when safe.
- Gameplay tiers for heavy, medium, solo and Goons-style bosses.
- Strict stronghold placement for heavy bosses such as Kaban and Glukhar.
- Excluded zones for unsafe/unfair locations.

1.0.6 highlights:

- Fixed 100% boss chance handling.
- Fixed weighted roll bias when total chances exceeded 100%.
- Confirmed Kaban + Glukhar can spawn together at Customs Fortress / ZoneScavBase.
- Woods ZoneDepo excluded by default.
- Low-level Ground Zero disabled by default; Ground Zero high-level remains enabled.

Requires SPT 4.0.13.
